# 💰 AI Powered Debt Relief & Financial Recovery Platform

An AI-powered web application that helps borrowers analyze their financial health, assess debt risk, receive personalized debt recovery recommendations, and generate professional debt negotiation letters using Google Gemini AI.

---

## 📌 Project Overview

Managing debt can be challenging without proper financial guidance. This platform uses Artificial Intelligence to evaluate a user's financial condition and provide personalized recommendations for financial recovery.

The application calculates financial metrics such as Debt-to-Income (DTI) Ratio, Financial Health Score, and Risk Level, then suggests suitable repayment strategies. It also generates AI-powered debt negotiation letters to help users communicate effectively with financial institutions. :contentReference[oaicite:1]{index=1}

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/AI-Powered-Debt-Relief-Platform.git
```

### Move into the Project Folder

```bash
cd AI-Powered-Debt-Relief-Platform
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```

## 📊 Workflow

1. Enter borrower financial details.
2. Validate user inputs.
3. Calculate financial metrics.
4. Determine financial health score.
5. Classify debt risk level.
6. Recommend suitable debt recovery strategies.
7. Generate AI-powered debt negotiation letter.
